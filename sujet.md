# Projet de Génie Logiciel - Première partie

**Durée:** 5 séances (15h)

## Introduction

Pour vous mettre en conditions réelles de développement d'une application (presque) comme en entreprise, vous allez devoir développer une application de suivi d'activités sportives (course à pied et cyclisme), similaire à la plateforme Strava.

Le but est de vous faire appliquer toutes les notions vues en cours:

- Conception d'une architecture Objets
- Mise en pratique des méthodes agiles pour organiser l'équipe
- Mise en pace de programmation défensive
- Développement des tests
- Documentation
- Utilisation de Git

## Consignes de Lancement

### Prérequis

- **Java 17** installé
- **Python 3** installé (pour le serveur frontend)
- Un terminal/ligne de commande
- Git installé sur votre ordinateur et compte Github créé
- Un IDE, au choix VScode, Eclipse ou Intelij

### Lancement du frontend

- `docs/frontend.md` : lancement du frontend, technologies utilisées, vérifications et dépannage

### Lancement des tests

- `docs/tests.md` : configuration JUnit, lancement des suites de tests

Bon travail !
